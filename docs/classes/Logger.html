Class Logger

Utility class to help with collecting and monitoring logs in your Sourc applications. Creating one is easy:

let my_logger = new Logger().withName("Your name here").withStrategy(() => {});

You can now use it like so:

my_logger.ERROR("Your message here");

Or of course any variant of TRACE, DEBUG, INFO, WARN, ERROR, or FATAL. You won't see anything happen though, until you register a LogHandler. A default one we ship is the ConsoleLogStrategy, that logs messages to the web console with colors.
