Class TextEditor

The heart and soul of Sourc. The TextEditor class is the gateway into your application. Begin by creating one, making sure to have any sort of HTMLElement ready:

let editor = new TextEditor(element);

You can access the TextEditor.LOGGER to enable or disable anything you want there now.

Next add any plugins you want to it (the core one is already loaded):

editor.registerPlugin(myPlugin);

See SourcPlugin for more info on creating plugins.

Now initialize the editor:

editor.initialize();

The element will become content-editable, and becomes a Sourc editor!
